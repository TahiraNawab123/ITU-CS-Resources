# ITU-CS-Resources

## Overview

This repository is designed for students pursuing degrees in Computer Science, Software Engineering, and Artificial Intelligence. It serves as a comprehensive resource for academic development, showcasing essential tools, methodologies, and knowledge areas crucial for success in these fields.

## Folder Structure

```
ITU-CS-Resources/
├── Algorithms/
│   ├── Sorting/
│   │   ├── bubble_sort.py
│   │   └── quick_sort.py
│   ├── Data_Structures/
│   │   ├── linked_list.py
│   │   └── binary_tree.py
├── Projects/
│   ├── Project1/
│   └── Project2/
├── Research/
│   ├── Papers/
│   ├── Articles/
│   └── Conferences/
└── Misc/
    ├── Resources/
    └── Tools/
```

## Guidelines for Use

- **Algorithms**: Implementations of common algorithms used in computer science.
- **Projects**: Practical applications of concepts learned.
- **Research**: Valuable resources for further academic exploration.
- **Miscellaneous**: Additional tools and resources that assist in academic pursuits.

This README aims to provide clarity and structure to the repository, ensuring students can find vital information quickly and efficiently.
