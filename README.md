# 📚 ITU CS Resources  
Centralized repository for **ITU (Information Technology University)** Computer Science past papers, assignments, labs, quizzes, midterms, and finals across core courses: **Programming Fundamentals (PF)**, **Data Structures & Algorithms (DSA)**, and **Object-Oriented Programming (OOP)**.

Organized by course and professor (**Sir UBS**, **Sir Sarfaraz**) for easy access during exam prep and assignments.

## 🚀 Quick Start  
1. Clone the repo:  
   \`\`\`bash  
   git clone https://github.com/TahiraNawab123/ITU-CS-Resources.git  
   cd ITU-CS-Resources  
   \`\`\`  
2. Open `ITU-Past-Papers/[COURSE]/[PROFESSOR]/` folders.  
3. Use PDF/DOCX viewers (e.g., LibreOffice, Adobe Reader) for documents.  
4. Search by course/professor/assignment/lab for specific resources.

## 💡 Features  
- **Comprehensive Coverage**: 100+ documents across 3 courses.  
- **Well-Organized**: Hierarchical folders for quick navigation.  
- **Ready-to-Use**: Direct downloads, no setup needed.  
- **Exam-Ready**: Past quizzes, mids, finals for practice.
